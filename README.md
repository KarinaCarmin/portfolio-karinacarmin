### 🔍 [Reputational Risk Analysis Agent](./reputational-risk-analysis)
> `Python` · `OpenAI API` · `Pandas` · `Web Search`

Script that automates reputational analysis of individuals or companies using AI with real-time web search. Reads a CSV of entities, queries news outlets, NGOs and social media, classifies findings by risk level (High / Medium / Low) and exports results to Excel and JSON.

**Highlights:**
- Segmented search across 3 source types: news media, NGOs and social networks
- Automatic classification by category: Corruption, Money Laundering, Fines, Complaints, and more
- Duplicate removal and consolidated export to Excel (4 sheets) + JSON
- Retry and error handling for API failures

**Highlights:**

- Data extraction (date, amount, links) and Excel report generation
